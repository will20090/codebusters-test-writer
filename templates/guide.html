User Guide

This brief guide should cover everything you need to know to build competition tests and practice tests! (wip)

General Tips

  • Tests autosave as you work. Look for the Saved indicator in the top bar.
  • Wait until you see the Saved indicator before refreshing or closing out of the tab.
  • This website will not work for any devices using iOS. The PDF viewer isn't supported by Apple's built in browser PDF viewer.

Test Builder

Used to build full tests for competitions with each question being fully customizable.

Tournament Setup

Click the Tournament Setup + Timed tab to configure (It should open up automatically when you first create a test):

  • Tournament name, division, and date
  • Timed question plaintext, point value, and signal phrase
  • Writer names
  • Cover image (jpg or png both work)

Adding Questions

  1. Pick a cipher and type (Decode, Crib, Extract, etc.)
  2. Enter the plaintext
  3. Fill in key fields for that cipher (keyword, shift, block size, etc.)
  4. Click Add Question
The question text is auto-generated. You can edit it in the text box below the key fields before adding, or click the question later to edit it.
If a question is invalid, a red error message should pop up telling you what to fix.

Managing Questions

  • Click any question to edit it in place
  • Drag questions to reorder them
  • × removes a question
  • Randomize Order shuffles the list
  • Bonus questions are marked with ★ and tracked automatically in the scoring table

Preview & Download

  • The previews should automatically update after each edit
  • Click Refresh Preview to manually compile and view the PDF on the right
  • Toggle between Test and Answer Key tabs
  • Download Test PDF / Download Key PDF to save
To download the PDFs, use the buttons at the top bar of the website, not the ones built-in to the PDF viewer. Using the PDF viewer buttons will not save your tournament info in the PDF name

Sharing

Click Share on any test to give other users edit access. You can remove their access at any time. Shared tests appear in the dashboard marked "Shared with you."

Edit History

The Edit History tab logs every change made to a test — who made it, when, and what changed. Click any entry to expand a before/after diff.

Practice Test Builder

Generates mixed practice tests automatically. Quotes are drawn from a database of ~40,000 English quotes and ~6,000 Spanish quotes, filtered by length for each cipher.

Creating a Test

  1. Pick a cipher and type, configure any key options
  2. Set a count for how many of that cipher to generate
  3. Click + Add to Queue
  4. Repeat for as many cipher types as you want

Generating

Click Generate Practice Test to generate all queued questions at once. A progress bar tracks generation. Failed questions are skipped with a warning in the console (Press Ctrl + Shift + I).

*Homophonic Decode Extra Info

For Homophonic Decode questions, you can control how much of the keyword is revealed to solvers:

Letters GivenDifficultyWhat solvers see
4 (default)—Full keyword, e.g. BACK
2 or 3EasyA random consecutive chunk, e.g. AC or BAC
2 or 3HardRandom letters in alphabetical order, e.g. A, C, and K
These options are ignored for Crib-type Homophonic questions.

Block Size

For ciphers that support block size: 0 or blank preserves word spacing in the ciphertext. Any positive integer groups letters into fixed-size blocks, hiding word boundaries.
